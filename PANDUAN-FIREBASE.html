🔥 Cara Setup Firebase

Ikuti 5 langkah di bawah ini untuk mengaktifkan sistem login sungguhan di website AI-Mossy kamu. Proses ini gratis dan hanya butuh sekitar 10–15 menit.

1
Buat Project Firebase

Buka console.firebase.google.com dan login pakai akun Google kamu.

Klik tombol "Add project" (atau "Tambahkan project").

  1. Beri nama project, misalnya: ai-mossy
  2. Google Analytics → boleh di-disable, klik Continue
  3. Klik "Create project" dan tunggu selesai
💡 Satu project Firebase bisa dipakai gratis untuk ribuan user (paket Spark/Free).
2
Daftarkan Web App

Setelah project dibuat, di halaman overview klik ikon </> (Web).

  1. Beri nama app, misalnya: ai-mossy-web
  2. Jangan centang "Firebase Hosting" (tidak diperlukan)
  3. Klik "Register app"

Setelah itu Firebase akan menampilkan firebaseConfig — ini yang kamu butuhkan di langkah 4.

💡 Kalau jendela config sudah tertutup, kamu bisa melihatnya lagi di: Project Settings → General → Your apps.
3
Aktifkan Metode Login

Di menu kiri Firebase Console, pilih Authentication → tab Sign-in method.

Aktifkan dua metode berikut:

  1. Email/Password → klik → toggle Enable → Save
  2. Google → klik → toggle Enable → pilih email support kamu → Save
⚠️ Untuk login Google, kamu perlu menambahkan domain website kamu ke Authorized domains (tab ketiga di Authentication). Saat development di localhost sudah otomatis diizinkan.
4
Salin Config ke login.html

Dari Firebase Console, copy firebaseConfig yang tampilannya seperti ini:

// Contoh tampilan firebaseConfig dari Firebase: const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "ai-mossy.firebaseapp.com", projectId: "ai-mossy", storageBucket: "ai-mossy.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123" };

Buka file login.html dengan text editor (Notepad, VS Code, dll.), cari bagian ini:

// ✏️ GANTI BAGIAN INI DENGAN CONFIG FIREBASE KAMU ↓↓↓ const firebaseConfig = { apiKey: "GANTI_apiKey", authDomain: "GANTI_authDomain", projectId: "GANTI_projectId", storageBucket: "GANTI_storageBucket", messagingSenderId: "GANTI_messagingSenderId", appId: "GANTI_appId" };

Ganti semua nilai "GANTI_..." dengan nilai asli dari Firebase kamu. Simpan file-nya.

💡 Setelah disimpan dan dibuka di browser, banner kuning peringatan akan otomatis hilang.
5
Upload & Test

Upload semua file (index.html, login.html, panduan ini) ke hosting kamu (GitHub Pages, Netlify, dll.).

Buka halaman login.html di browser dan coba:

  1. Klik "Daftar" → isi nama, email, password → klik "Buat Akun Gratis"
  2. Cek apakah akun muncul di Firebase Console → Authentication → Users
  3. Coba login dengan akun yang baru dibuat
  4. Coba tombol "Lanjutkan dengan Google"
💡 Semua akun yang mendaftar akan otomatis tersimpan di Firebase Console → Authentication → Users. Kamu bisa hapus, suspend, atau reset password dari sana kapan saja.

🎉 Selesai!

Setelah langkah di atas selesai, sistem login AI-Mossy kamu sudah beneran aktif dengan database sungguhan.
Semua akun user tersimpan otomatis di Firebase — aman, gratis, dan scalable.
